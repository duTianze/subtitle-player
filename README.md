# Subtitle Player

subtitle-player is a java GUI Application for play subtitle independent.

## Required
jdk 17

## Usage

```shell
 ./mvnw spring-boot:run
```
up -> jump to pre subtitle

down -> jump to next subtitle

space -> pause
